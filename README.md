# passwordchecker
Checking password whether it is hacked or not using 'https://haveibeenpwned.com/ API'

To use it:

in CMD

type

(for window 10) -->
python3 -m pip install --upgrade requests

(Mac or Linux) -->
sudo pip install requests

After installing the package -->
python3 checkmypass.py (type your password without bracket) (you can enter multiple password after space)
